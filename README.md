# interaction-spark-app-
PURPOSE OF THE APP:
The Interaction Spark App is an Android application designed to help users maintain their social connections throughout the day.
The app provides meaningful interactive suggestions based on the time of day entered by the user (example: morning,mid-morning,night etc)

THE GOAL IS TO:
*Help users stay connected with their loved ones by providing quick results that enhance their experience and creativity.
*Build consistent communication habits.
*Make socislising quick and fun.

ATTRIBUTES:
*A simple layout for easy interaction
*Use of clean texts and colours for readability
*Input-based suggestion system
*Buttons for responsiveness
*Readable fonts and sizes 

DESIGN FACTORS:
*Clear contrast between text and background
*ConstraintLayout
*Full-width buttons
*Font sizes and styles

*Users are guided  with hints
*Clear feedback is provided
*Suggestions are displayed clearly
*Simple and easy to use
*Error handling for invalid/empty input

HOW IT WORKS:
*Enter a time of day using suggestions provided
*The app validates input
*If input is valid then a suggestion is displayed
*If input is invalid an error message is shown
*Reset button clears all fields
